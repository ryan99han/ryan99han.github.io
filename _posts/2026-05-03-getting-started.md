Welcome to my personal site. This is a minimal space for sharing notes on papers, thoughts on machine learning and AI, and general musings about the industry.

## How I'll use this

- **Paper notes**: Summaries and reflections on interesting ML research
- **Industry musings**: Thoughts on where the field is headed, lessons learned, etc.
- **Meta**: Notes about the site itself and how I'm approaching writing

## The setup

This site is built with Jekyll and hosted on GitHub Pages, which makes it trivial to add new posts—just write markdown and push. No complicated deployments needed.

Feel free to check back for updates, or follow the [RSS feed](/feed.xml).
